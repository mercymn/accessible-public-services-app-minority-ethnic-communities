# Accessible Public Services App for Minority Ethnic Communities

## Overview

This project presents a user-centred mobile application prototype designed to improve access to essential public services for minority ethnic communities.

Developed as part of the Advanced Human-Computer Interaction coursework at Heriot-Watt University, the project addresses barriers such as language differences, accessibility challenges, limited awareness of available services, and difficulties navigating digital systems.

The prototype incorporates multilingual support, voice-assisted interaction, emergency assistance features, service enquiries, and accessibility-focused design principles to support more inclusive access to public services.

## Problem Statement

Minority ethnic communities often experience barriers when accessing essential services due to language differences, digital exclusion, limited service awareness, and accessibility challenges.

This project explores how user-centred design approaches can be used to create a more inclusive and accessible digital experience that supports access to housing, health, and community services.

## Target Users

* Minority ethnic communities
* Individuals with limited English proficiency
* Users requiring accessible digital interfaces
* Individuals seeking information about public and community services

## Key Features

* Multilingual language selection
* Voice-assisted interaction
* Emergency support functionality
* Service enquiries and guidance
* Accessibility-focused navigation
* Inclusive user interface design
* Housing application support
* User notifications and status tracking

## Design Process

The prototype was developed using a user-centred design approach, including:

* Requirements gathering and analysis
* Literature review and user research
* Persona development
* Use case development
* Storyboarding and user journey design
* Low- and high-fidelity prototyping
* Accessibility-focused design decisions
* Usability evaluation and iterative refinement

## Evaluation

The prototype was evaluated using a mixed-methods approach combining:

* System Usability Scale (SUS)
* User observation
* Think-aloud evaluation
* User interviews

Evaluation findings indicated strong user satisfaction, positive perceptions of usability, and appreciation for the multilingual and accessibility-focused features. Feedback from participants informed recommendations for future improvements.

## Tools and Methods
